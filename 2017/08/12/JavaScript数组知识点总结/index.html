JavaScript数组知识点总结

什么是数组：

数组也是对象,比对象多了一个length属性。

每个对象都是由属性名和属性值组成。

我们看到的是数组的属性值，数组的属性名是从0开始的数字,我们把代表第几项的数字的属性名叫做数组的“索引”。

数组有一个length属性，代表数组的长度。

获取当前索引的属性值方法：arr[索引]。

获取数组的总长度：arr.length

数组中的length属性，是浏览器天生就给数组增加的，不是我们自己编写的属性，这个属性属于当前数组不可枚举的属性。

循环数组中的每一项

数组中常用方法

学习数组中常用的方法，我们需要按照四个维度去探讨

  • 方法的作用
  • 需要传哪些参数
  • 是否有返回值，返回值是什么
  • 通过此方法是否对原来数组产生改变

    关于数组的增加和删除方法

    1.push():
  • 作用：可向数组末尾添加一个或多个元素
  • 参数：向末尾增加的一个或多个元素
  • 是否有返回值：返回新的数组长度

  • 原数组是否改变：原数组改变

  • 扩展：不使用push,也想向数组末尾增加内容。

2.pop():

  • 作用：删除数组最后一个元素
  • 参数：没有参数
  • 是否有返回值：返回数组最后一个值

  • 原数组是否改变：原数组改变，数组长度减1

  • 扩展：不使用pop,也想删除数组末尾内容。

3.shift():

  • 作用：删除数组第一项
  • 参数：没有参数
  • 是否有返回值：返回数组第一项的值
  • 原数组是否改变：原数组改变，数组长度减1

4.unshift():

  • 作用：数组前端添加一项或多项
  • 参数：要添加的数字
  • 是否有返回值：返回添加后的新数组长度
  • 原数组是否改变：原数组改变。

5.splice():

  • 作用：既能实现删除，也能实现增加，还能实现修改
  • 参数：删除(start,delCount):第一项位置，要删除的项数
    插入(start,0,args):start起始位置，0要删除项数，args插入的项
替换(start,delCount,args):起始位置，要删除项数，插入的任意数
  • 是否有返回值：把删除的内容以一个新数组的方法返回
  • 原数组是否改变：原数组改变。

关于数组的截取和拼接

6.slice():实现数组的截取，在原来的数组中截取某一部分

  • 作用：从已有数组中返回规定的元素
  • 参数：(n,m),从索引n开始，找到索引为m处的数组(不包含m)
  • 是否有返回值：返回截取的数组
  • 原数组是否改变：原数组不变
    slice(n):从索引n开始一直找到数组的末尾
    slice(0),slice():把原来的数组克隆一份一模一样的新数组返回
    slice(-1);//[11]:从数组尾部开始算起的元素(-1是最后一个元素，-2是倒数第二个元素)

7.concat():

  • 作用：把两个数组拼接到一起
  • 参数：ary1.concat(ary2);参数为要拼接的数组
  • 是否有返回值：无返回值
  • 原数组是否改变：原数组不变
    ary1.concat():把ary1克隆一份一模一样的数组

不考虑原数组是否改变，克隆数组方法有：
ary1.concat():把ary1克隆一份一模一样的数组
ary1.slice();
ary1.slice(0);
ary1.splice(0);

可数组转换为字符串

8.toString():

  • 作用：把数组转换为字符串
  • 参数：无参数
  • 是否有返回值：无返回值
  • 原数组是否改变：原数组不变

9.join():

  • 作用：按照每一个分隔符，把数组中的每一项拼接成一个字符串
  • 参数：分隔符
  • 是否有返回值：无返回值
  • 原数组是否改变：原数组不变

扩展：eval：js中把字符串变为js表达式执行的一个方法

console.log(eval('23+43+32+11'));//109

把字组中的每一项进行相加求和

console.log(eval(ary.join('+')));

关于数组的排序和排列方法

10.reverse():

  • 作用：把数组倒序排列
  • 参数：无
  • 是否有返回值：返回倒序的数组
  • 原数组是否改变：原数组改变

11.sort():

  • 作用：给数组进行排序
  • 参数：无
  • 是否有返回值：返回排序后的数组
  • 原数组是否改变：原数组改变
    ary.sort():这样的话只能处理10以内的数字进行排序，因为它是按照UNICODE编码进行排序的。

由小到大排序方法：

ary.sort(function(a,b){
    return a-b;
})

只有在标准浏览器中兼容的方法，在IE6~8不兼容的方法

12.indexOf()

  • 作用：当前内容在数组中第一次出现位置的索引
  • 参数：索引值
  • 是否有返回值：如果数组中没有这一项，返回-1，如果有这一项，索引是几就返回几
  • 原数组是否改变：原数组不变
    indexOf用法：可以检证当前内容在数组中是否存在

13.lastIndexOf()

  • 作用：当前内容在数组中最后一次出现位置的索引
  • 参数：索引值
  • 是否有返回值：如果数组中没有这一项，返回-1，如果有这一项，索引是几就返回几
  • 原数组是否改变：原数组不变
    字符串中也有这两个方法，但是字符串的这两个方法兼容所有的浏览器。
    14.forEach()
  • 作用：遍历数组中的每一项
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：没有返回值
  • 原数组是否改变：原数组不变

15.map()

  • 作用：遍历数组中的每一项(map和forEach语法一样，但比forEach多加了一个把原来内容进行替换的功能)
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：返回每次函数调用的结果组成的数值
  • 原数组是否改变：原数组不变

map、forEach、for区别：

1. map速度比forEach快
2. map会返回一个新数组，不对原数组产生影响
   forEach不会产生新数组，对原数组也没有影响
3. map因为返回数组所以可以链式操作，forEach不能
4. map、forEach都比for要更好用

16.every()

  • 作用：遍历数组中的每一项,对数组中的每一项运行给定函数，如果该函数对每一项都返回true,则返回true
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：true或false
  • 原数组是否改变：原数组不变

17.some():

  • 作用：遍历数组中的每一项,如果该函数有满足条件的元素，则返回true
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：true或false
  • 原数组是否改变：原数组不变

18.filter()

  • 作用：遍历数组中的每一项,返回该函数会返回true的项组成的数组
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：返回符合条件组成的数值
  • 原数组是否改变：原数组不变

19.find()

  • 作用：遍历数组中的每一项,查找某个元素，返回true,则将当前遍历元素返回，不继续查找
  • 参数：函数中的形参：数组项的值item,该项在数组中的位置index,数组对称本身input
  • 是否有返回值：返回符合条件组成的元素
  • 原数组是否改变：原数组不变

归并方法

20.reduce()

  • 作用：迭代数组的所有项，然后构建一个最终返回值。从数组第一项开始，逐个遍历到最后
  • 参数：函数中的形参：prev:前一个值，cur:当前值，index:项的索引，array:数组对象
  • 是否有返回值：返回符合条件组成的数值
  • 原数组是否改变：原数组不变
